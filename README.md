# Ellära Study Hub

Study tool for KTH IF1330 (Electrical Principles), built spec-first during a 38-day exam prep sprint. Features spaced-repetition flashcards, RC/RL/RLC transient simulator, DC/AC circuit solver with nodal and mesh analysis, and auto-generated practice problems.

**Live demo:** [ellara-study-hub.vercel.app](https://ellara-study-hub.vercel.app)

**Stack:** Next.js 14, TypeScript, Tailwind CSS, math.js, Recharts, KaTeX

See [SPEC.md](./SPEC.md) for full project specification.

## Development

```bash
npm install
npm run dev
```
